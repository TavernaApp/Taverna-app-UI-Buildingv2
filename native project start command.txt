Sabse pehle, terminal ya command prompt kholen.
Project ke liye ek naya directory banayein aur usmein jayein:
mkdir myProject
cd myProject
Ab, React Native CLI ko global install karein (agar aapne pehle se install nahi kiya hai):

npm install -g react-native-cli
Project ko initialize karein:
react-native init MyReactNativeApp
Project directory mein jaayein:
cd MyReactNativeApp
Development server ko start karein:
npx react-native start
Ek new terminal window/tab open karein aur project ko run karein:
For Android:
npx react-native run-android
For iOS:
npx react-native run-ios
